## 🔧 Projects I Maintain

### 📁 [MiXplorer · Google Drive Mirror](https://github.com/nOneCode4u/mixplorer-google-drive)

> Automated daily mirror of MiXplorer and all official add-ons from H. Parsa's Google Drive to GitHub.

[![Total Downloads](https://img.shields.io/github/downloads/nOneCode4u/mixplorer-google-drive/total?style=flat-square&logo=android&logoColor=white&color=3DDC84)](https://github.com/nOneCode4u/mixplorer-google-drive/releases)
[![Latest](https://img.shields.io/github/v/release/nOneCode4u/mixplorer-google-drive?style=flat-square&logo=github&color=0969da&label=Latest)](https://github.com/nOneCode4u/mixplorer-google-drive/releases/latest)
[![Daily Update](https://img.shields.io/github/actions/workflow/status/nOneCode4u/mixplorer-google-drive/daily_update.yml?style=flat-square&label=Daily%20Update&logo=githubactions&logoColor=white)](https://github.com/nOneCode4u/mixplorer-google-drive/actions)

MiXplorer is a fast, ad-free file manager for Android — but its APKs are only distributed via an XDA thread and a private Google Drive. This mirror brings them to GitHub so Obtainium users can track and auto-update without the Play Store.

**Mirrors:** MiXplorer · MiX Archive · MiX Codecs · MiX Encrypt · MiX Image · MiX PDF · MiX Tagger

### 📰 [Kishor Publisher](https://github.com/nOneCode4u/kishor-publisher)

> Automated Telegram channel uploader for Kishor — a free Marathi monthly magazine for children.

[![Python](https://img.shields.io/badge/Python-3776AB?style=flat-square&logo=python&logoColor=white)](https://python.org)
[![Telegram](https://img.shields.io/badge/Telegram-2CA5E0?style=flat-square&logo=telegram&logoColor=white)](https://t.me/KishorMarathiMonthlyMagazine)
[![GitHub Actions](https://img.shields.io/badge/GitHub%20Actions-2088FF?style=flat-square&logo=githubactions&logoColor=white)](https://github.com/nOneCode4u/kishor-publisher/actions)

Detects new issues on ebalbharati.in daily, schedules uploads with one-hour gaps, and publishes each as a named PDF with a cover thumbnail. Zero manual work after setup.
